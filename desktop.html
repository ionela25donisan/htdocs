1. Prezentare Generală Proiect

Titlu Proiect: Sistem de Gestiune Integrată a Apartamentelor

Aplicație web pentru administratorii de proprietăți care permite gestionarea completă a portofoliilor de apartamente, inclusiv urmărirea financiară (facturi și chirii), programarea și urmărirea întreținerii, stocarea documentelor juridice, și gestionarea informațiilor chiriașilor.

Obiective Principale

  • Centralizare datelor - Toate informațiile despre apartamente și chiriași într-un singur sistem
  • Eficiență administrativă - Reducerea timp consumat în gestionarea manuală a dosarelor
  • Transparență financiară - Urmărire ușoară a veniturilor și cheltuielilor
  • Conformitate legală - Stocarea și organizarea documentelor importante
  • Comunicare ușoară - Gestionare eficientă a informațiilor chiriașilor

Utilizatori Principali

  • Administratori de proprietăți
  • Proprietari de apartamente
  • Personal de management

2. Funcționalități și Module

Module Principale

1. MODUL CHIRIAȘI

Scop: Gestionare informații chiriași

Funcționalități:

  • Adăugare / editare / ștergere chiriași
  • Vizualizare informații personale (nume, email, telefon)
  • Istoric contracte de închiriere
  • Data început și finalizare contract
  • Status actual (activ/inactiv)
  • Contacte de urgență
  • Referințe plăți anterioare

Date Principale: ID Chiriaș, Nume, Email, Telefon, Adresa, Alte Informații Contact, Data Început Lease, Data Sfârșit Lease, Status Chiriaș

2. MODUL APARTAMENTE

Scop: Administrare informații apartamente

Funcționalități:

  • Listă completă de apartamente
  • Detalii apartament (adresă, număr cameră, suprafață)
  • Chiriaș actual alocat
  • Istoric locatari
  • Status ocupare (liber/ocupat)
  • Număr contact urgență
  • Note speciale despre proprietate

Date Principale: ID Apartament, Adresă, Număr Apartament, Suprafață, Număr Camere, Etaj, Chiriaș Actual, Data Ocupare, Status

3. MODUL FACTURI

Scop: Gestionare facturi utilități și servicii

Funcționalități:

  • Creare facturi (apa, gaz, electricitate, întreținere)
  • Setare dată scadență
  • Urmărire status plăți
  • Generare rapoarte lunare
  • Notificări pentru plăți restante

Date Principale: ID Factură, ID Apartament, Tip Factură, Sumă, Dată Scadență, Status Plată

4. MODUL CHIRII

Scop: Urmărire plăți chirii

Funcționalități:

  • Înregistrare plăți chirii lunare
  • Urmărire întârzieri
  • Calcul penalități
  • Rapoarte venituri
  • Notificări pentru plăți

Date Principale: ID Chirie, ID Apartament, ID Chiriaș, Lună, Sumă Chirie, Status Plată, Dată Plată

5. MODUL ÎNTREȚINERE

Scop: Programare și urmărire reparații

Funcționalități:

  • Programare lucrări de întreținere
  • Urmărire status reparații
  • Alocare tehnicieni
  • Înregistrare costuri
  • Rapoarte întreținere

Date Principale: ID Întreținere, ID Apartament, Descriere, Status, Prioritate, Dată Programare, Cost Estimat

6. MODUL DOCUMENTE

Scop: Stocare documente juridice

Funcționalități:

  • Încărcare documente (contracte, acte)
  • Organizare pe categorii
  • Căutare documente
  • Acces securizat
  • Backup automat

Date Principale: ID Document, ID Apartament, Tip Document, Nume Fișier, Cale Fișier, Dată Încărcare
