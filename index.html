🎯 Prezentare Generală

Acesta este un prototip care prezintă planificarea pentru un sistem de gestiune apartamente, cu module pentru facturi, chirii, întreținere, documente și chiriași.

Obiective

  • Centralizare informații apartamente și chiriași
  • Gestionare facturi și chirii rapidă
  • Planificare mentenanță și stocare documente
  • Interfață prietenoasă și vizuală pentru mobil și desktop

Tehnologii

  • HTML5, CSS3, JavaScript
  • PHP
  • MySQL
  • Apache / HTTP

🧩 Funcționalități și Module

Chiriași

Adăugare, editare, vizualizare chiriași, contracte și istoric plăți.

Facturi

Gestionare facturi utilități, scadențe și rapoarte de plăți.

Chirii

Urmărire plăți chirii, restanțe și rapoarte venit.

Întreținere

Programare și monitorizare lucrări de mentenanță.

Documente

Stocare contracte și acte cu căutare și status expirare.

🗄️ Schema Bazei de Date

Schema este construită pe tabele separate pentru a păstra datele relaționale curate și ușor de extins.

Tenants: id, nume, email, telefon, contract start/end, status
Apartments: id, adresă, camere, suprafață, status, tenant_id
Invoices: id, apartment_id, tip, sumă, data scadență, status
Rents: id, apartment_id, tenant_id, lună, sumă, status
Maintenance: id, apartment_id, descriere, data, status, cost
Documents: id, apartment_id, tip, nume fișier, path, expirare

🏗️ Arhitectură Tehnică

Aplicația este gândită modular: frontend static pentru mockup și backend PHP/MySQL pentru implementare reală.

Structură fișiere

  • index.php - Dashboard real
  • css/ - Stiluri
  • js/ - Funcționalitate
  • php/ - CRUD și API
  • Specs/ - Documentație

Principii

  • Separare logică / prezentare
  • Interfață responsivă
  • Securitate și validare
  • Filtrare și căutare avansată
