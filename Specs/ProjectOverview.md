# Project Overview

This project is an apartment management system designed to handle various aspects of property management including invoices, rents, maintenance, documents, and tenants.

## Objectives

- Provide a comprehensive tool for property managers to manage apartments efficiently.
- Include features for financial tracking, maintenance scheduling, document storage, and tenant management.

## Scope

- Web-based application using HTML, CSS, JS, PHP, MySQL.
- User-friendly interface for easy navigation and data management.
