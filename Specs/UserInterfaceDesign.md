# User Interface Design

## Pages

- Dashboard: Overview of key metrics.
- Tenants: List and manage tenants.
- Apartments: List and manage apartments.
- Invoices: Manage invoices.
- Rents: Manage rent payments.
- Maintenance: Schedule and track maintenance.
- Documents: Upload and view documents.

## Design Principles

- Clean, modern design.
- Responsive layout.
- Easy navigation.
